# NIGHTWATCH BEYOND // PROJECT 210-165

A high-performance, tactical training HUD built for the **Nothing Phone (2a)**. 
Designed for low-light gym environments with an OLED-optimized interface.

## 🎯 Core Mission
- **Target Descent:** 210 lbs → 165 lbs
- **Program Duration:** 8-Week Macrocycle
- **Protocol:** Automated Strength Progression Engine

## 🛠 Features
- **PWA Ready:** Install directly to home screen for offline use.
- **Neural Link:** Automated lift-weight calculation based on previous rep counts.
- **Tactical UI:** Red-on-Black HUD with scanline effects.
- **Data Privacy:** 100% local storage. Your data never leaves your device.

## 🚀 Deployment
1. Upload `index.html` to the root.
2. Enable **GitHub Pages** in Settings.
3. Open the provided link and "Add to Home Screen" on your mobile device.
